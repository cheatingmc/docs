## Overview

Checking stock levels is essential before making purchases. This guide shows you how to view available Minecraft accounts, their prices, and current stock levels through our Discord bot.

## Quick stock check

### Basic command

Use this simple command to see all available accounts:

```
/stock
```

This will display:
- All available account types
- Current prices in credits
- Stock levels for each type
- Quality information

## Advanced stock checking

### Check specific account type

To get detailed information about a specific account type:

```
/stock unban
/stock ban
/stock cookie
```

This provides:
- Detailed pricing information
- Current stock count
- Quality specifications
- Delivery estimates

### Stock alerts

Set up notifications for low stock or restocks:

```
/stock-alerts on
/stock-alerts unban
/stock-alerts off
```

<Info>
**Stock alerts:** Get notified when stock levels change or when your preferred account types are restocked.
</Info>

## Stock-related commands

### Complete command list

```
/stock                    # View all available stock
/stock [type]            # View specific account type
/stock-alerts on         # Enable stock notifications
/stock-alerts off        # Disable stock notifications
/stock-alerts [type]     # Set alerts for specific type
/balance                 # Check your credit balance
/history                 # View purchase history
```

### Command examples

```
/stock unban             # Check Unban AccessToken stock
/stock ban               # Check Ban AccessToken stock
/stock cookie            # Check Cookie account stock
/stock-alerts unban      # Get notified when Unban stock changes
```
