## Account formats explained

### AccessToken format

```
email:password:access_token

Example:
user@example.com:password123:eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
```

**Components:**
- **Email:** Microsoft account email address
- **Password:** Account password
- **Access Token:** JWT token for authentication

### Cookie format

```
email:password:cookie_data

Example:
user@example.com:password123:session_id=abc123; auth_token=xyz789; user_prefs=pref1,pref2; csrf_token=csrf123
```

**Components:**
- **Email:** Microsoft account email address
- **Password:** Account password
- **Cookie Data:** Full session cookies and tokens
